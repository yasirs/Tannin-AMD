# Data Directory

This directory contains all data used and generated by the project, organized by source and type.

## Directory Structure

*   **`external/`**: Public datasets downloaded from repositories like GEO, Figshare, etc.
    *   **`geo/`**: Gene Expression Omnibus datasets.
        *   **`GSE135092/`**: AMD RNA-seq cohort (N=537).
        *   **`GSE29801/`**: AMD Microarray cohort (N=293).
        *   **`GSE99287/`**: AMD RPE ATAC-seq Atlas.
        *   **`GSE129964/`**: RPE Serum Starvation Timecourse.
    *   **`perturbseq/`**: Large-scale CRISPR screens (Replogle et al. 2022).
*   **`RPE_cells/`**: Internal RPE bulk RNA-seq data (H2O2/PRG4 perturbations).

## Data Management Policy

*   **Git:** Raw data files (fastq, bam, large csvs) are **ignored** by git to keep the repository light.
*   **Reproduction:** Use the scripts in `code/external-validation/` to re-download or re-process these datasets.
